# Leon - HTML & CSS Agency Template

This is a hands-on implementation of the first template from Elzero Web School's **"HTML & CSS Template One 2021"** playlist. The project is a clean, responsive agency landing page called "Leon," focused on translating a professional design into semantic code while practicing core front-end skills.

**🚀 Live Demo:** [View Deployed Site](https://haithamabudraz.github.io/elzero-html-css-template-one/)

---

## 📖 About the Project

The goal was to solidify my understanding of HTML5 and CSS3 by building a complete, multi-section website from a design mockup. The template—named **"Leon"**—is styled as a minimalist creative agency portfolio. My implementation covers the full build process: structuring semantic HTML, creating a pixel-perfect layout with CSS, and ensuring responsiveness for different screen sizes.

The design features the following sections:
*   **Header:** Fixed navigation with a responsive hamburger menu.
*   **Landing Section:** A bold, centered introduction.
*   **Features:** A three-column grid highlighting key services.
*   **Services:** Detailed service listings with icons and side text.
*   **Portfolio:** Project cards with hover effects.
*   **About:** A creative, asymmetrical layout.
*   **Contact:** A clean call-to-action and contact info.
*   **Footer:** Simple and professional sign-off.

---

## ✨ Features

*   **Pixel-Perfect Design:** Faithful translation of the provided PSD/AI template into HTML & CSS.
*   **Fully Responsive:** `Mobile-first` design approach ensures it works flawlessly on phones, tablets, and desktops.
*   **Clean Semantics:** Uses modern, meaningful HTML5 elements (`<header>`, `<nav>`, `<section>`, etc.) for great accessibility and SEO.
*   **CSS Grid & Flexbox:** Heavily utilizes both Grid and Flexbox modules for robust, modern layouts.
*   **Interactive Components:** CSS-only hover effects, a responsive hamburger menu, and smooth scrolling.
*   **Font Awesome Integration:** Uses scalable vector icons for the services section.

## 💡 What I Learned

Building this project was a fantastic pathway from theory to practice. Key takeaways include:

*   **CSS Layout Mastery:** Gained deep, practical experience using **CSS Grid** for page-level and card layouts, and **Flexbox** for aligning components within sections.
*   **Responsive Design Techniques:** Mastered media queries, fluid typography (`clamp()`, `rem`), and responsive image handling.
*   **Custom Components:** Built, from scratch, a CSS-only responsive navigation menu with a toggleable hamburger icon, and designed custom component shapes (like the distinctive "L" image in the About section).
*   **Design-to-Code Workflow:** Improved my ability to analyze a static design and systematically structure it into reusable HTML and CSS components.
*   **Basic State Management:** Added simple and elegant interactive states with the `:hover` pseudo-class on cards and buttons.
